# PhysioCare - Clinic Management System

A simple, free-to-use, and localized management platform for physiotherapy clinics.

## Fitur Utama
- **Dashboard**: Statistik pendapatan, biaya, dan laba.
- **Kelola Pasien**: Pencarian dan pendaftaran pasien baru.
- **Jadwal Janji Temu**: Manajemen kunjungan pasien.
- **Pelacakan Keuangan**: Catat pendapatan dan biaya operasional.
- **Bahasa & Mata Uang**: Full Bahasa Indonesia & Rupiah (IDR).
- **Offline & Installable**: Berfungsi sebagai PWA (Progressive Web App).

## Cara Menjalankan Secara Lokal

1. Instal dependensi: `npm install`
2. Jalankan server: `npm run dev`
3. Buka [http://localhost:3000](http://localhost:3000) di browser Anda.

## Cara Mengetes di HP
### 1. Menggunakan Jaringan Lokal (WiFi)
- Pastikan HP dan komputer Anda berada di jaringan WiFi yang sama.
- Cari alamat IP komputer Anda (misal: `192.168.1.xx`).
- Di browser HP Anda, buka `http://[ALAMAT-IP-ANDA]:3000`.

### 2. Instalasi PWA
- Setelah membuka aplikasi di browser HP (Chrome di Android atau Safari di iOS):
- **Android**: Klik menu (titik tiga) lalu pilih "Instal Aplikasi" atau "Tambahkan ke Layar Utama".
- **iOS**: Klik tombol Share (kotak dengan panah ke atas) lalu pilih "Tambahkan ke Layar Utama".

### 3. Deployment Gratis (Rekomendasi)
- Anda bisa men-deploy aplikasi ini secara gratis ke **Vercel** atau **Netlify** agar bisa diakses dari mana saja melalui internet.
